### 1) 줄바꿈   
__ex1>__   
```
  document.write(내용 + "<br>")
```

__ex2>__: 문자열 여러줄 출력   
```
  return "문자\n문자\n문자";
```
<br>

### 2) 공백 (space bar)   
```
  return 변수a + " " + 변수b  // a b
```
<br>

### 3) 따옴표   
__ex1>__: 문자열 따옴표   
```
  return "\"Back End\"";  // 따옴표 앞 역슬래쉬
                          // "Back End"
```
<br>
<hr>

### 4) 문자열   
__ex>__: __```split()```__   
```
var str1 = "Hello World";

str1.split(" ");   // 공백 기준으로 나눈 후 배열,  [Hello, World]
```
<br>

* __```변수명.shift```__: 배열 앞에 데이터 삭제
