### 1) 무지개
#### (1) 문제
* 100px의 정사각형을 사용하여 세로로 긴 무지개를 만드시오 (빨강, 주황, 노랑, 초록, 파랑)

* 색깔: 빨강 (#FF0000), 주황(#FF8811), 노랑(#FFFF00), 초록(#00FF00), 파랑(#0000FF)
<br>

__HTML >__
```
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>무지개</title>
    <link rel="stylesheet" href="index.css" />

    <style>
        * {
            margin: 0;
            padding: 0;
        }
        #red {
            background-color: #FF0000;
        }
        #orange {
            background-color: #FF8811;
        }
        #yellow {
            background-color: #FFFF00;
        }
        #green {
            background-color: #00FF00;
        }
        #blue {
            background-color: #0000FF;
        }
    </style>
  </head>
  
  <body>
    <div id="red"></div>

    <div id="orange"></div>

    <div id="yellow"></div>

    <div id="green"></div>

    <div id="blue"></div>
  </body>
</html>
```

<br>

__CSS >__
```
div {
    height: 100px;
    width: 100px;
}
```
