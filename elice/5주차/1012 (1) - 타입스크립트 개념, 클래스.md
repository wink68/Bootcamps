### (1) 타입스크립트
* 자바스크립트의 상위 집합

   * ES6 < TpyeScript 2.0

   * 타입스크립트를 쓰기 위해선 자바스크립트로 변환해주는 컴파일 과정이 필요

<br>

### (2) 타입스크립트를 쓰는 이유
* (자바스크립트의) 동적 타입을 정적으로 선언하기 위해

__Ex >__
```
let a : number;  // 숫자값만 할당 가능

a = 1;
a = 'b';   // 에러 발생
```

<br>
   
* 타입 유추를 통한 타입 제어가 가능하다

__Ex >__: 자바스크립트
```
const sum = (a, b) => {
   return a + b
}

sum(1, "2")  // 12
```

__Ex >__: 타입스크립트
```
const sum = (a: number, b: number) => {
   return a + b
}

sum (1, 2) // 3
```
<br>

* 자바스크립트 컴파일 시점에 오류를 포착할 수 있다
