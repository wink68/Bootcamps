## 2. 자바스크립트 문제집1   
### 1) 문자열 함수 만들기 (2번)
```
function repeatString(x, y) {
    let str = "";
    for (let i=1; i <=y; i++) {
        str += x;
    }
    return str;
};
```

### 2) 배열 오름차순 정렬하기 (3번)
```
function sortStringArray (arr) {
    return arr.sort();
}
```

<br>

### 3) 배열 오름차순 정렬 및 뒤집기 (4번)
```
function reverseStringArray (arr) {
    return arr.sort().reverse();
}
```
