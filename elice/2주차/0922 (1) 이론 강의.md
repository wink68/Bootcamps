### (1) 웹 APIs
* 수동적으로 기능을 만들지 않고 누군가 이미 만들어진 기능들 (Application Programming Interfaces)

* web APIs

   * DOM APIs - 웹 페이지 요소들을 조작 가능

   * Network APIs - 서버와 통신하는 기능

   * Graphics APIs - 그래픽 관련 기능

   * Audio/Video APIs - 멀티미디어 관련 기능 (오디오 재생 중지 등)
